MarkNotes

tags: Apps

MarkNote is an app to help you to take notes with Markdown.

FEATURES

  • Embed photos and images into notes

  • Live preview while writing
  • Auto save
  • markdown highlighting to assist editing
  • Syntax highlighting for embedded code blocks
  • Import any plain text file as note
  • Export notes into HTML format

You have full control to your data. And the app will never track you or collect your privacy.
